# 心庭（こころにわ）設計書一式

AIハッカソン2026予選2ndラウンド用、テーマ「ZEN」の提出作品設計書。

## 📄 ファイル構成

| ファイル | 内容 | 使い方 |
|---------|------|--------|
| **CLAUDE.md** | プロジェクトルール・制約事項 | リポジトリのルートに置き、Claude Codeに自動的に参照させる |
| **SPEC.md** | 詳細仕様書 | 実装前に一度全部読む、実装中に参照 |
| **PROMPTS.md** | 段階別プロンプト集 | Claude Codeに順番に投げていく |
| **README.md** | これ | 全体俯瞰 |

## 🚀 使い方

### ステップ1: リポジトリ準備（黒木さん）
```bash
mkdir zen-garden && cd zen-garden
# この3ファイルをコピーする
cp CLAUDE.md SPEC.md PROMPTS.md ./
git init && git add . && git commit -m "初期設計書"
# GitHubにpush、Pages有効化
```

### ステップ2: Claude Code起動
```bash
cd zen-garden
claude  # Claude Codeを起動
```

### ステップ3: 順次プロンプトを投げる
PROMPTS.md の Phase 0 → 1 → 2 → 3 の順に、プロンプトを1つずつ投げる。
各フェーズ完了後、動作確認してから次へ。

## 🎯 このアプリの勝ち筋

### 審査員巻き込み型デモ
1. 福岡さん本人に岩を置いてもらう
2. AIが即座に庭を解説
3. **福岡さんの3Dアバター召喚**で会場沸く
4. 記念撮影でSNS拡散誘発

### 福岡さんのツボ直撃
- ① 一般人がパッと見て面白い → 3D操作の直感性
- ② テック×ポップカルチャー → 枯山水×AI
- ③ 触って驚く体験 → アバター召喚のサプライズ
- ④ ストーリー・世界観 → 老師キャラと和テイスト

### 過去の失敗を回避
- ✅ AI感が濃い（Geminiで生成テキスト多用）
- ✅ 横スクロールじゃない（3D枯山水）
- ✅ 画像生成に依存しない（テクスチャ事前用意）
- ✅ 効果音で手触り向上
- ✅ 複雑すぎない（岩を置くだけ）
- ✅ フォールバック完備（AI落ちても動く）
